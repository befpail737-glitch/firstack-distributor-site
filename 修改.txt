修复所有页面的导航栏（最关键）
复制标准<header>： 打开您本地的 index.html 文件，完整地复制从 <header class="main-header"> 开始，到 </header> 结束的所有代码。它看起来是这样的：
code
Html
<header class="main-header">
    <div class="container">
        <a href="/" class="logo-link">
            <img src="/images/logo/logo.svg" alt="YourCompanyName Logo - Authorized Firstack Distributor" class="logo-img">
        </a>
        <nav class="main-nav">
            <ul class="nav-links">
                <li><a href="/" class="active">Home</a></li>
                <li><a href="/products.html">Products</a></li>
                <li><a href="/solutions.html">Solutions</a></li>
                <li><a href="/support.html">Support</a></li>
                <li><a href="/news.html">News</a></li>
                <li><a href="/about.html">About Us</a></li>
                <li><a href="/contact.html" class="cta-button nav-cta">Contact Us</a></li>
            </ul>
        </nav>
        <button class="mobile-nav-toggle" aria-label="Toggle Navigation">
            <span class="hamburger"></span>
        </button>
    </div>
</header>
批量替换： 现在，您需要打开以下所有HTML文件，找到它们各自的（不完整的）<header>...</header>部分，然后用您刚刚复制的这段完整的代码彻底替换掉它。

微调 active 状态（可选，但建议）： 为了让当前页面的导航高亮，您需要手动修改每个文件里导航栏的 class="active"。
例如，在编辑 about.html 时，您需要把 active 从 Home 移到 About Us 上：
code
Html
<!-- 在 about.html 文件里 -->
...
<li><a href="/">Home</a></li>
...
<li><a href="/about.html" class="active">About Us</a></li>
...
对 products.html, solutions.html 等所有主页面都做同样的操作。文章详情页可以不设置active状态，或者让其父级导航高亮（例如，在一篇解决方案文章里，让 "Solutions" 导航高亮）。
